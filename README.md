## Features

- Next.js 16 App Router with TypeScript (React 19)
- Tailwind CSS styling + custom animations
- Framer Motion interactions
- AI chat powered by OpenRouter with usage/cost telemetry
- PostHog analytics
- Mobile-first, responsive design
